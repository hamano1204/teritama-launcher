# Suika Text Expander (すいかテキストエクスパンダー)

<p align="center">
  <img src="suika_icon.png" width="128" height="128" alt="Suika Text Expander Icon" />
</p>

WPF で開発された Windows 用の軽量でスタイリッシュな定型文挿入（テキストエクスパンダー）アプリケーションです。
システムトレイに常駐し、設定したグローバルホットキーからいつでもポップアップメニューを呼び出し、選択した定型文をアクティブなエディタに素早く自動入力（貼り付け）できます。

---

## 主な機能

### 1. システムトレイ常駐 & ポップアップメニュー
* タスクバーを占有せず、システムトレイ（通知領域）に常駐します。
* グローバルホットキー（デフォルト: `Ctrl + Shift + S`）を押すと、マウスカーソルの位置に定型文のポップアップメニューが表示されます。
* ポップアップ外をクリックするか、ESCキーを押すことで瞬時にメニューを閉じることができます。

### 2. 直感的でスムーズなキーボード/マウス操作
* **キーボード操作**: 矢印キー（`↑` `↓`）で項目を選択、`Enter` で決定（テキスト挿入）、`Esc` や `←` でメニューを閉じる / 親フォルダへ戻る。
* **マウス操作**: マウスホバーで瞬時にサブフォルダを展開。ホバー時のハイライト色とキーボード選択時のハイライト色（ブルー）が統一されています。

### 3. 多彩な定型文ツリー構造
* フォルダによる階層化（サブフォルダ対応）。空のフォルダがある場合は「（空）」と分かりやすく表示されます。
* 項目間の区切り線（セパレーター）をサポート。
* リスト内のアイコンもフラットなデザインで統一され、フォルダ（📁）、定型文（📄）、セパレーター（➖）が視覚的に区別しやすくなっています。

### 4. 高度なエディター機能
* 定型文・フォルダ・セパレーターの追加、編集、削除。
* ドラッグ＆ドロップによる階層や順序の直感的な並べ替え。
* データのインポート / エクスポート機能（JSON形式）。

### 5. 動的プレースホルダー
* 定型文の中に `{{now:yyyy/MM/dd HH:mm:ss}}` のような日時フォーマットを指定することで、挿入時に自動的に現在日時に置換されます。

### 6. その他の設定
* グローバルホットキーのカスタマイズ（修飾キー Ctrl/Shift/Alt/Win + 任意のキー）。
* Windows ログイン時の自動起動（スタートアップ登録）の有効 / 無効切り替え。

## 動作環境

* **OS**: Windows 10 / 11
* **フレームワーク**: .NET 10 (WPF)

---

## インストール・ビルド方法

### 開発環境でのビルド手順

1. **リポジトリのクローン**:
   ```bash
   git clone https://github.com/hamano1204/suika-text-expander.git
   cd suika-text-expander
   ```

2. **ビルドと実行**:
   .NET 10 SDK がインストールされている環境で、以下のコマンドを実行します。
   ```bash
   dotnet build
   dotnet run
   ```

## 使い方

1. アプリケーションを起動すると、システムトレイ（タスクバーの右下）にスイカのアイコンが表示されます。
2. テキストを入力したいエディタ（メモ帳、ブラウザ、VS Code など）をアクティブにします。
3. ホットキー（デフォルト: `Ctrl + Shift + S`）を押します。
4. ポップアップメニューから挿入したい定型文を矢印キーまたはマウスで選択し、決定します。
5. 設定や定型文の編集を行いたい場合は、システムトレイのアイコンをダブルクリックするか、右クリックメニューから「編集 / 設定」を選択してエディターウィンドウを開きます。

---

## 各種設定データの保存先
設定および定型文データは、以下のディレクトリに `data.json` として保存されます。
* `%APPDATA%\SuikaTextExpander\data.json`
