# Teritama Launcher (テリタマランチャー)

<p align="center">
  <img src="teritama_icon.png" width="128" height="128" alt="Teritama Launcher Icon" />
</p>

WPF で開発された Windows 用の軽量でスタイリッシュなアプリケーションランチャーです。
システムトレイに常駐し、設定したグローバルホットキーからいつでもポップアップメニューを呼び出し、登録したショートカットやアプリ、URLを素早く起動できます。

---

## 主な機能

### 1. システムトレイ常駐 & ポップアップメニュー
* タスクバーを占有せず、システムトレイ（通知領域）に常駐します。
* グローバルホットキー（デフォルト: `Ctrl + Shift + Win + R`）を押すと、マウスカーソルの位置にランチャーのポップアップメニューが表示されます。
* ポップアップ外をクリックするか、`Esc` キーを押すことで瞬時にメニューを閉じることができます。

### 2. 直感的でスムーズなキーボード/マウス操作
* **キーボード操作**: 矢印キー（`↑` `↓`）で項目を選択、`Enter` で決定（アプリケーション/URL起動）、`Esc` や `←` でメニューを閉じる / 親フォルダへ戻る。サブメニューは `→` キーで展開。
* **マウス操作**: マウスホバーで瞬時にサブメニューを展開。ホバー時のハイライト色とキーボード選択時のハイライト色（ブルー）が統一されています。
* **UI/UXの最適化**: 親メニューとサブメニューが隙間なくぴったり吸着して表示されるよう、DPI対応の位置計算ロジックが組み込まれています。

### 3. 多彩なツリー構造と見やすいデザイン
* フォルダによる階層化（サブフォルダ対応）。空のフォルダがある場合は「（空）」と分かりやすく表示されます。
* メニューの端から端までしっかり伸びる、美しくクリーンな区切り線（セパレーター）をサポート。
* ファイルやアプリケーションの「システムアイコン」を自動で取得・表示する機能（`SystemIconService`）を搭載し、実行時スレッド間で安全にキャッシュ（Frozen処理）して読み込みます。

### 4. 高度なエディター機能
* ランチャー項目・フォルダ・セパレーターの追加、編集、削除。
* ドラッグ＆ドロップによる階層や順序の直感的な並べ替え。
* 外部ファイル（アプリケーションやショートカットなど）をエディターのツリーにドラッグ＆ドロップして、直接インポートすることが可能。
* データのインポート / エクスポート機能（JSON形式）。

### 5. その他の設定
* グローバルホットキーのカスタマイズ（修飾キー Ctrl/Shift/Alt/Win + 任意のキー）と、重複や無効なキーを防ぐバリデーション機能。
* Windows ログイン時の自動起動（スタートアップ登録）の有効 / 無効切り替え（`Environment.ProcessPath` を使用した正確なパス指定）。

## 技術スタック・アーキテクチャ

* **OS**: Windows 10 / 11
* **フレームワーク**: .NET 10 (WPF)
* **主なパッケージ**:
  * `CommunityToolkit.Mvvm` (MVVM パターン)
  * `H.NotifyIcon.Wpf` (システムトレイ制御)
  * `Newtonsoft.Json` (設定データのシリアライズ)
* **ファイル構成と役割**:
  * `App.xaml / App.xaml.cs`: アプリケーションの初期化、ホットキー制御、ポップアップ表示管理。
  * `MainWindow.xaml / MainWindow.xaml.cs`: 非表示ウィンドウ、タスクトレイアイコンの設定。
  * `Models/SnippetNode.cs`: ランチャー項目（Snippet / Folder / Separator）のデータモデル。
  * `Models/AppConfig.cs`: ホットキー設定や自動起動設定の定義。
  * `Services/NativeMethods.cs`: Win32 API（ホットキー登録やマウスフック等）の集約定義。
  * `Services/SnippetManager.cs`: データの読込・保存。保存失敗時のエラーハンドリングや、破損時の自動 `.bak` バックアップ復元機能を備えています。
  * `Services/HotkeyService.cs`: グローバルホットキーの登録と監視。
  * `Services/KeyboardHookService.cs` / `MouseHookService.cs`: 低レベルフックによるキーボード・マウス操作監視。
  * `Services/LaunchService.cs`: アプリケーションやURLのプロセス起動。
  * `Services/StartupService.cs`: Windowsスタートアップ（レジストリ）への登録・解除。
  * `Services/SystemIconService.cs`: ファイル拡張子やパスからシステムアイコンを取得しキャッシュするサービス。

## 各種設定データの保存先

設定および登録データは、以下のディレクトリに `data.json` として保存されます。
* `%APPDATA%\TeritamaLauncher\data.json`

※ データ保存時に万が一ファイルが破損した場合は、自動的に作成される `%APPDATA%\TeritamaLauncher\data.json.bak` バックアップファイルから復元を試みます。

## インストール・ビルド方法

.NET 10 SDK がインストールされている環境で、以下の手順に従ってビルドおよび実行を行ってください。

### 通常のビルドと起動
```powershell
# ビルド
dotnet build

# 実行
dotnet run --project TeritamaLauncher.csproj
```

### クリーンビルドの手順
キャッシュや古いビルド成果物を完全に削除してビルドし直したい場合は、以下のコマンドを実行します。
```powershell
# クリーンアップ（bin/objフォルダの削除）
dotnet clean

# 再ビルド
dotnet build
```

## 使い方

1. アプリケーションを起動すると、システムトレイ（タスクバーの右下）にテリタマのアイコンが表示されます。
2. ホットキー（デフォルト: `Ctrl + Shift + Win + R`）を押します。
3. ポップアップメニューから起動したいアプリケーションやURLを矢印キーまたはマウスで選択し、決定します。
4. 設定やランチャー項目の編集を行いたい場合は、システムトレイのアイコンをダブルクリックするか、右クリックメニューから「定型文編集」を選択してエディターウィンドウを開きます。
